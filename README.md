# tsotra

Plateforme web pour les **stages non rémunérés** et le **volontariat structuré**.

## Idée

Centraliser les opportunités encadrées et accessibles dès la première
expérience pour les étudiants et les débutants. Pas de job board classique,
pas de freelance — uniquement des missions transparentes.

## Stack

- Django 5 + Django REST Framework
- SQLite (prototype), PostgreSQL prévu en production
- React (à venir)
